Behavioural Design

The focus of this section of the Style Guide is ensuring user needs for usability and function are supported. This includes interactive components, navigation patterns, and forms that make mental health resources accessible and intuitive for all users.

Link Styles

This section demonstrates how link states support accessibility and visual clarity. Clear link design helps users understand interactive elements and navigate the site with confidence.

Interactive Buttons

Buttons provide clear calls-to-action with visual feedback for user interactions. This section demonstrates feedback (on hover) and feedforward (labels/icons) principles.

Button Variants

Interactive States Demo

Hover over and click the button below to see interaction feedback:

Design Features: Buttons use color psychology to indicate importance (primary > secondary > outline). Hover and active states provide immediate feedback, while consistent padding ensures comfortable touch targets.

Button Sizes

Usage Guidelines: Medium buttons are used for most actions. Large buttons draw attention to primary calls-to-action, while small buttons are used for secondary actions in tight spaces.
