What's in this website?

Work Phase 2 focused on understanding user needs through comprehensive design research. This phase involved analyzing client requirements, conducting user interviews to develop personas and user stories, and evaluating the project through accessibility and ethics frameworks. Key deliverables included client needs analysis, user research interviews, accessibility assessment following WCAG guidelines, and an ethics analysis applying the ACS Code of Professional Conduct. The research findings inform design decisions and ensure the community platform serves diverse users while maintaining ethical standards and accessibility compliance.
